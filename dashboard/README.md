# 电商用户行为分析 Dashboard

基于 **Streamlit** + **Plotly** + **Polars** 构建的交互式数据可视化平台，服务于拼多多数据分析师求职项目。

---

## 环境要求

- Python 3.9+
- 推荐内存：≥ 8GB（数据集约 1.7GB，Polars 内存占用远低于 Pandas）

---

## 安装依赖

```bash
cd E:\NewWorkProject\PDD\pdd-data-analyst-project\dashboard
pip install -r requirements.txt
```

依赖列表：
- `streamlit` — 交互式 Web 应用框架
- `plotly` — 交互式图表库
- `polars` — 高性能 DataFrame 库（替代 Pandas 处理大文件）
- `pandas` — Polars `to_pandas()` 导出及 CSV 下载所需

---

## 启动方式

```bash
streamlit run app.py
```

启动后，浏览器会自动打开 `http://localhost:8501`。

---

## 功能介绍

| 模块 | 说明 |
|------|------|
| **核心指标卡片** | 顶部展示总用户数、总 PV、购买转化率、用户复购率 4 个 KPI |
| **侧边栏筛选器** | 支持日期多选、行为类型多选、小时范围滑块、周末/工作日筛选 |
| **流量趋势分析** | DAU 折线图、各行为类型每日趋势、小时分布柱状图 |
| **转化漏斗** | Plotly 漏斗图：PV → Fav → Cart → Buy，附带各环节转化率表格 |
| **用户价值分析** | 按购买频次分层饼图、用户活跃度分布、每日复购率趋势 |
| **商品分析** | 热销商品 TOP10、热销类目 TOP10、商品转化效率散点图（PV vs Buy） |
| **数据探索** | 展示前 1000 行数据表格（支持排序/筛选），并提供 CSV 下载按钮 |

---

## 技术亮点

- **Polars 高性能读取**：29M 行 CSV 秒级加载，内存占用低
- `@st.cache_data` 数据缓存：避免重复读取 1.7GB 文件，切换筛选条件时响应迅速
- **Plotly 交互式图表**：支持悬停提示、框选缩放、图例筛选
- **低饱和度暖色调配色**：统一使用橙、棕、珊瑚色系，视觉舒适
- **响应式布局**：`wide` 模式 + `use_container_width=True`，适配不同屏幕
- **数据采样与预聚合**：商品散点图超过 5000 点时自动采样，表格仅展示前 1000 行

## 注意事项

- 若数据文件路径变更，请修改 `app.py` 顶部的 `DATA_PATH` 常量。
- 首次加载数据需要约 5~15 秒（取决于磁盘速度），请耐心等待缓存完成。
- 若出现内存不足，可关闭其他应用或考虑在服务器端运行。
