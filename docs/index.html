如何验证本项目

Level 1
1 分钟

查看 README 顶部的实时 CI 徽章，点击可查看每次 lint / sql-lint / docker-build 的完整日志。

Level 2
3 分钟

docker compose -f docker-compose.superset.yml up -d
一个命令启动 BI 看板，浏览器打开即可看到真实数据。

Level 3
10 分钟

make all
从 raw CSV 到最终报告完整跑通，预期 0.4 秒处理 29M 行。
